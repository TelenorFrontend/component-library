What are Molecules?
  • Collections
  • Elements
  • Form elements
    • Input fields
    • Toggles
Molecules consist of one or more nested rules, but each of them must not be more than an atom itself. For example, using icons or buttons in molecules is valid, but using other molecules is not.
To mark an element as molecule, annotate it with the @molecule tag, followed by the name of the component.

Media
Show example
 
Copy markup
A media element that comprises several diferent child classes. Examples can be found in the media column structure.
  • tn-media: main class
    • tn-media__heading: media heading.
    • tn-media__image: an image that spans the width of the media element and retains its aspect ratio.
    • tn-media__caption: centered text.
    • tn-media__content: genral-purpose container for any other element.

The tn-media element fills the parent containers width and scales in height to fit its content.
